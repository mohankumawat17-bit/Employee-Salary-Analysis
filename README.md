# Employee Salary Dashboard (SQL + Power BI)

## Project Overview
This project analyzes employee salary data to understand department-wise cost distribution
and key salary metrics. SQL is used for data extraction and aggregation, and Power BI is used
to build an interactive dashboard for reporting and decision-making.

The dashboard provides a clear view of total salary cost, average salary, employee count,
and department-wise salary contribution.

---

## Tools & Technologies
- MySQL
- SQL
- Power BI
- Excel

---

## Dataset
The dataset contains employee-level information with the following fields:
- Employee ID
- Name
- Department
- Salary

---

## SQL Work
SQL was used to:
- Retrieve employee data using SELECT queries
- Perform aggregations using SUM, AVG, and COUNT
- Analyze department-wise salary distribution
- Apply business logic using GROUP BY and JOIN
- Prepare summarized data for reporting

All SQL queries used in this project are included in the SQL script file.

---

## Power BI Dashboard
The Power BI dashboard includes:
- KPI cards for:
  - Total Salary
  - Average Salary
  - Employee Count
- Bar chart showing Salary by Department
- Donut chart showing Salary % Contribution by Department
- Employee-level details table
- Department slicer for interactive analysis

---

## Key Insights
- IT department contributes the highest share of total salary cost.
- Average salary in IT is higher compared to HR.
- Department-level filtering helps analyze workforce cost efficiently.

---

## Project Outcome
This project demonstrates an end-to-end data analysis workflow:
Excel / Database → SQL → Power BI Dashboard

It highlights practical skills in data analysis, reporting, and dashboard creation
using industry-relevant tools.
