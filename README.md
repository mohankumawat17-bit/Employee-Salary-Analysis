Employee Salary Analysis Dashboard

Tools Used:
- MySQL
- Power BI
- SQL

Project Description:
This project analyzes employee salary data by department.
SQL was used to clean, join, and summarize the data.
Power BI was used to create an interactive dashboard with KPIs,
department-wise salary distribution, and filters.

Key Features:
- Total, Average Salary KPIs
- Employee Count
- Department-wise Salary Analysis
- Interactive slicer for department filtering
